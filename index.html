GitHub Guide for Web Projects

Memory tool for myself on how to use GitHub to store, manage, and showcase your web development projects

Why Use GitHub for Your Projects?

Version Control

Track changes to your code over time and revert to previous versions if needed.

Portfolio Building

Showcase your projects to potential employers or clients.

Collaboration

Work on projects with others and contribute to open source.

Backup & Cloud Storage

Keep your code safe in the cloud and accessible from anywhere.

How to Set Up a GitHub Repository

1 Create a GitHub Account

Go to github.com and sign up for a free account.

2 Install Git

Download and install Git from git-scm.com.

3 Configure Git

Set up your username and email in Git:

git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

4 Create a New Repository

On GitHub, click the "+" icon in the top right and select "New repository".

5 Initialize Git in Your Project

Navigate to your project folder and run:

git init

6 Connect to GitHub

Add your GitHub repository as the remote origin:

git remote add origin https://github.com/your-username/your-repo-name.git

7 Add, Commit, and Push

Add your files, commit changes, and push to GitHub:

git add . # Adds all files
git commit -m "Initial commit"
git push -u origin main

Pro Tip: GitHub Pages for Free Hosting

GitHub offers free hosting through GitHub Pages. Simply enable it in your repository settings, and your website will be available at https://your-username.github.io/repository-name.
